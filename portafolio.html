Mi Introducción al HTML, primer desafío en academia LATAM, generar un HTML de estructura básica para crear un CV.
Mi introducción a CSS; se debía crear un HTML y utilizar características CSS para darle estilo al HTML.
Alineado con FLEX; el objetivo era a travéz de una maqueta, utilizar propiedades CSS para lograr crearla tal cual como en la maqueta.
Bootstrap; se inicia el uso de Bootstrap para trabajar más rapidamente con recursos previamente creados, modificables y mejorables según necesidad.
Termina, GIT y GITHUB; se aplica todo lo aprendido para realizar una página con las características solicitadas, logrando compartirlo en GITHUB a través de la terminal.
GtHub Pages; en base a todo lo aprendido, se lleva a cabo la remodelación del primer CV creado en HTML, aplicando todos los conocimientos adquiridos y publicándolo en GitHub Pages.
